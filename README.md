# 账单自动解析上传到Notion

这个服务的作用是通过解析账单邮件，然后自动的上传到notion中，减少人为手动输入账单信息。

在notion中，自动根据数据生成图表。

# 使用教程

1. 复制Notion 模板
2. 创建Notion 集成
3. 发送账单邮件
4. 解析邮件并发送到notion
